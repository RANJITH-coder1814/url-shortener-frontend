# url-shortener-frontend
A simple web application that allows users to enter a long URL and generate a shortened URL (demo version). Users can also copy the shortened link easily.
# URL Shortener Frontend

A simple URL shortener frontend project built using HTML, CSS, and JavaScript.

## Features
- Enter a long URL
- Generate a short URL
- Copy shortened URL to clipboard
- Simple and responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository
2. Open index.html in your browser
